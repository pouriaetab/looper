# Looper (for /market/looper/)

## Overview
Market analysis and visualization dashboard using Streamlit. Real-time charts, technical analysis, portfolio tracking.

## Tech Stack
- **Framework**: Streamlit
- **Language**: Python 3.10+
- **Data**: pandas, numpy, plotly, yfinance

## Running
```bash
./run.sh
```

Access: `http://127.0.0.1:8501`

## Features
- Market overview
- Technical analysis (SMA, RSI, MACD)
- Portfolio tracking
- Backtesting

## Data Source
Default: yfinance (15-min delay)
Alternative: Alpaca API
